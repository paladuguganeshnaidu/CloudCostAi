Thank you for your interest in contributing to CloudCostAI.

Quick start:

- Fork the repository.
- Create a feature branch: `git checkout -b feat/my-change`.
- Run tests: `python -m pytest -q`.
- Commit your changes and open a PR against `main`.

Coding style:
- Please follow existing project patterns and keep changes focused.
- Run formatting before commit (e.g., `black .`).

Security:
- Do not commit secrets or `.env` files. Use environment variables for configuration.

Thank you — maintainers will review your PR.
